# Payment Options for Kids' Projects

## Why Payment Processors Need Your Info

Payment processors like Stripe need:
- Phone number (for security/verification)
- Business/legal name
- Tax ID or SSN (for tax reporting)
- Address verification
- You must be **18+ years old** (legal requirement)

This is required by law for processing real money payments.

## How to Use Stripe Test Mode

1. Go to https://dashboard.stripe.com/register
2. Sign up with any email (doesn't need phone verification)
3. You'll automatically be in "Test mode"
4. Get your test API keys (starts with `sk_test_` and `pk_test_`)
5. Use test card: `4242 4242 4242 4242` (won't charge real money!)

**Test mode is FREE and requires NO phone number or verification!**
