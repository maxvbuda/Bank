## Features

- **Transfers**: Send Red Diamonds to other users
- **Transaction History**: Track all your banking activities

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## How to Use

1. **Register**: Create a new account (you'll start with 1000 Red Diamonds!)
2. **Login**: Sign in with your credentials
3. **Deposit**: Add Red Diamonds to your account
4. **Withdraw**: Take Red Diamonds out of your account
5. **Transfer**: Send Red Diamonds to other users by their username
6. **View History**: See all your transactions

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: SQLite
- **Security**: bcrypt for password hashing
